### 1. System Requirements
* Software Environment:** This program runs on **Python 3.11.9**.
* Hardware Requirements: No specialized hardware is required. The code can be executed on a standard personal laptop or desktop computer.

### 2. Installation and Quick Start

To run the code, please follow these steps:
1. Open the Code: Open the project in an IDE such as **Cursor** or **VS Code**.
2. Install Dependencies: Install all the required software dependencies (e.g., via "pip install -pandas").
3. Configure Paths: Modify the input file paths within the script to point to your local data.
4. Execute: Run the script. The generated output data files will be automatically saved in the same directory as the input files.

### 3. Example Data

* The file/folder named "Example..csv" "Blank.csv" contains the sample data for demonstration.
